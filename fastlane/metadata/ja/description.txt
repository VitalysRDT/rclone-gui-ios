Rclone GUIは、パーソナルクラウドストレージ管理の定番ツールrcloneのための、ネイティブかつオープンソースのiOSクライアントです。Amazon S3、Backblaze B2、Cloudflare R2、Storj、Wasabi、Dropbox、Google Drive、Microsoft OneDrive、pCloud、Mega、Box、Filen、Internxt、Drime、SFTP、WebDAV、その他70以上のサービスに、iPhoneから仲介なしで直接接続できます。

【プレミアムサブスクリプションが必要】
Rclone GUIのすべての機能(リモートの管理、転送、「ファイル」App連携、PhotoSync、crypt復号、ストリーミングなど)をご利用いただくには、プレミアムサブスクリプションが必要です。

• 7日間無料トライアル後、月額1.99ユーロまたは年額19.99ユーロ
• 自動更新方式で、「設定」→「Apple ID」→「サブスクリプション」からいつでも解約できます
• オンボーディングを除き、有効なサブスクリプションなしでは機能を利用できません

【crypt-first暗号化(プレミアム)】
アプリ全体のUIは、rcloneの"crypt"暗号化を中心に設計されています。暗号化されたリモートは紫色の鍵アイコンで識別され、ファイル名はその場で復号されるため実際のフォルダ構造がそのまま確認でき、コンテンツも再生中にその場で復号されるので、平文の状態で第三者のサーバーを経由することは一切ありません。暗号処理の中核はNaCl secretbox(XSalsa20-Poly1305)で、すべてrcloneによりローカルで実行されます。

【iOS「ファイル」App連携(プレミアム)】
File Provider拡張機能により、crypt暗号化されたリモートを含むすべてのリモートがAppleの「ファイル」Appに表示されます。「ファイル」Appから直接PDFを開いたり、App間でドキュメントをドラッグ&ドロップしたり、動画や音楽を全体をダウンロードせずにストリーミング再生したりできます。

【スマート写真同期(プレミアム)】
新しく撮影した写真や動画を、お好みのクラウドへ自動的に同期します。アプリ使用中は転送速度を適応的に調整し、バックグラウンドでも実行され、ネットワークが切断された場合も安全に再開されます。

【ガイド付き追加ウィザード(プレミアム)】
ステップバイステップのウィザードが、すべてのrcloneバックエンドに対応しています。リモートの種類ごとに最適化された入力項目、OAuth(Drive、OneDrive、Dropbox)向けのチュートリアル、その場でのバリデーションを提供します。「認証情報の取得方法」ガイドや、リモートセレクター(crypt保管庫、alias、unionなど向け)が、設定作業をさらにシンプルにします。既存のrclone.confも一度の操作でインポートでき、シークレット情報はiOSキーチェーンへ自動的に移行されます。

【プライバシーを重視した設計】
- アナリティクスなし、トラッカーなし、サードパーティ製SDKなし
- バックエンドサーバーは存在せず、アプリはお使いのクラウドサービスと直接通信します
- Face ID / Touch IDで設定をロック
- 設定情報は保存時に暗号化(ChaCha20-Poly1305 + iOS Keychain)
- OAuth認証情報はiOSキーチェーンに保存され、UserDefaultsには一切保存されません
- 厳格なATS(App Transport Security)を適用し、ネットワーク例外なし

【主な機能(すべてプレミアムサブスクリプションに含まれる)】
- 70以上のrcloneバックエンドに対応
- 並列転送、一時停止・再開、転送キュー
- 通信量に応じた適応的な帯域制限(モバイル通信 / Wi-Fi)
- range requestsに対応した動画・音楽ストリーミング
- QuickLookによるプレビュー内蔵
- フォルダ内での検索・並べ替え・フィルター
- 詳細な転送統計
- デバッグ用にエクスポート可能なローカルログ

【サブスクリプションの条件】
Rclone GUI Premiumサブスクリプションは、各期間の終了時に自動的に更新されます(月額1.99ユーロまたは年額19.99ユーロ)。更新料金は、期間終了の24時間前までにご利用のApple IDに請求されます。サブスクリプションの管理や自動更新の解除は、「設定」→「Apple ID」→「サブスクリプション」からいつでも行えます。
