Rclone GUI is a native, open-source iOS client for rclone — the reference tool for managing your personal cloud storage. Connect Amazon S3, Backblaze B2, Cloudflare R2, Storj, Wasabi, Dropbox, Google Drive, Microsoft OneDrive, pCloud, Mega, Box, SFTP, WebDAV, and 60+ other services straight from your iPhone, with no middleman.

CRYPT-FIRST ENCRYPTION (Premium)
The whole interface is built around rclone "crypt" encryption: encrypted remotes are tagged with a purple lock, filenames are decrypted on the fly so you see the real tree, and file content is decrypted during playback without ever transiting in clear through a third-party server. The cryptographic core uses NaCl secretbox (XSalsa20-Poly1305), executed locally by rclone.

iOS FILES INTEGRATION (Premium)
A File Provider extension exposes every remote inside Apple's Files app, encrypted ones included. Open a PDF straight from Files, drag and drop documents between apps, and stream video or audio without downloading the full asset.

SMART PHOTO SYNC (Premium)
Automatically sync new photos and videos to the cloud of your choice. Adaptive throttling while you're using the app, background execution, clean resume after a network drop.

GUIDED ADD-REMOTE WIZARD (Premium)
A step-by-step wizard covers every rclone backend: per-backend forms, tutorials for OAuth providers (Drive, OneDrive, Dropbox), instant validation. Import an existing rclone.conf in one tap, with automatic secret migration to the iOS Keychain.

PRIVACY BY DESIGN
- No analytics, no tracker, no third-party SDK
- No backend server: the app talks directly to your cloud services
- Face ID / Touch ID to lock the configuration
- Configuration encrypted at rest (ChaCha20-Poly1305 + iOS Keychain)
- OAuth tokens in the iOS Keychain, never in UserDefaults
- Strict ATS, no network exception
