Novità della versione 2.1:
- Modificare un remoto: una nuova schermata «Gestisci remoti» consente di correggere un'impostazione o riautorizzare un account OAuth scaduto senza ricreare tutto — le password e i token esistenti restano nascosti.
- Remote Lens: dai un'occhiata a un file remoto senza scaricarlo — anteprima dell'immagine con i suoi dati EXIF e prima pagina dei PDF.
- Eliminazione riparata: eliminare un file o una cartella, o spostarlo nel cestino, funziona di nuovo.
